Valuation Rules

All values are entered in kLocal currency and converted to the selected target currency through the FX table.

Gross value (target currency) = Local value x effective FX rate
Effective reduction = Open: target x progress; Implemented: target; Abandoned: 0
Effective cost = Open/Abandoned: target x progress; Implemented: target
Occurrences = Recurrent annual: End year - Start year + 1; all other modes: 1
Total residual value = (Gross - Effect + Cost) x Occurrences when status = Open, otherwise 0
Total provision = Total residual value x provision %

The provision percentage is adjustable per risk (25% by default; 20% observed on some lines).
